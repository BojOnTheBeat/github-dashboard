# github-dashboard

## Requirements

- `$ pip install flask`
- `$ pip install flask-WTF`
- `$ pip install PyGithub`
- Obtain a github api access token.


## To run
- `$ export FLASK_APP=dashboard.py`
- `$ export GITHUB_TOKEN=<your access token here>`
- `$ flask run`
